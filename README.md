## 🛠️ Getting Started

1. **Clone the repository**

   ```
   git clone https://github.com/GilbertHx/pokemon.git
   cd pokemon
   ```
   
2. **Install dependencies**

   ```
   npm install
   ```

3. **Install dependencies**

   ```
   npm start
   ```

4. **View in browser**

   Open http://localhost:3000
